# マイページ分析結果

## 参考サイト
https://lme.jp/manual/user_settting/

## 確認された機能

### 1. 管理者設定（マイページ）
- **登録情報の確認・編集**
  - 管理者名
  - メールアドレス
  
- **パスワードの変更**
  - 新しいパスワードへの変更機能
  
- **設定を更新**
  - 変更内容を保存するボタン

### 2. システム設定関連（スクリーンショットから確認）
- システム管理関連
- エルメシステム設定
- LINE公式アカウント設定
- エラーメッセージ
- 見積書発行 ★
- 新規アカウント追加
- データコピー
- 決済システム連携設定

## 実装する機能

### マイページ
1. **ユーザー情報表示・編集**
   - 管理者名
   - メールアドレス
   - プロフィール画像

2. **アカウント設定**
   - パスワード変更
   - **請求書発行機能** ★
     - 請求書一覧表示
     - 請求書ダウンロード（PDF）
     - 請求書の検索・フィルター
   
3. **通知設定**
   - メール通知設定
   - プッシュ通知設定

4. **セキュリティ設定**
   - 二段階認証
   - ログイン履歴

5. **プラン・支払い情報**
   - 現在のプラン表示
   - 支払い履歴
   - 請求書管理
