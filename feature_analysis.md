# 機能分析メモ

## 1. チャット機能の要件

### 主要機能
- **友だち一覧**：左側に友だちリスト表示
- **絞り込み機能**：全ての友だち、未確認、確認済み、非表示中、グループ
- **タグ・対応ステータスによる絞り込み**
- **検索機能**：LINE名、システム表示名で検索
- **クイックアクション**：ブックマーク、対応ステータス変更、確認状況変更、非表示・ブロック、タグ編集
- **送信ユーザー名管理**：スタッフ名の設定・変更・削除
- **チャット画面**：メッセージ送受信、履歴表示
- **友だち情報表示**：基本情報、タグ、リッチメニュー、ステップ配信状況
- **メモ機能**：友だちごとのメモ作成・編集・削除

## 2. 通知設定の要件

### 通知媒体
- スマートフォンアプリ
- ChatWork（チャットワーク）
- PCデスクトップ

### 通知タイミング
- リアルタイム
- 15分～24時間の間で選択可能

### 通知項目
- 友だち追加
- メッセージ受信
- フォーム回答
- 予約登録
- 商品注文
- その他各種イベント

## 3. QRコードアクション（流入経路分析）の要件

### 主要機能
- **QRコード生成**：友だち追加用のQRコード・URL発行
- **稼働対象設定**：新規友だちのみ / 全ての友だち
- **アクション設定**：メッセージ送信、タグ付与、リッチメニュー切り替え
- **フォルダ管理**：QRコードアクションの整理
- **データ分析**：
  - URL読み込み人数
  - 友だち追加・ブロック解除人数
  - アクション稼働人数
- **詳細データ**：
  - 数値情報（期間別）
  - 友だち一覧
  - 分岐詳細
- **CSVエクスポート**：データのダウンロード
- **有効期限設定**：期間限定のQRコード

## 4. スタッフのロール別デフォルト権限

### 副管理人（ほぼ全権限）
- 友だち管理：全機能
- メッセージ：全機能
- フォーム・予約：全機能
- 商品・注文：全機能
- 分析：全機能
- マーケティング：全機能
- チャット：全機能
- 連携：閲覧のみ
- システム設定：スタッフ管理以外

### 運用者（日常業務）
- 友だち管理：友だちリスト、タグ管理
- メッセージ：一斉配信、ステップ配信、自動応答、テンプレート
- フォーム・予約：全機能
- 商品・注文：商品管理、注文管理
- 分析：友だち分析、メッセージ分析
- マーケティング：流入経路管理
- チャット：1:1チャット
- 連携：なし
- システム設定：なし

### サポート（顧客対応）
- 友だち管理：友だちリスト（閲覧のみ）
- メッセージ：テンプレート（閲覧のみ）
- フォーム・予約：フォーム回答、予約管理
- 商品・注文：注文管理（閲覧のみ）
- 分析：なし
- マーケティング：なし
- チャット：1:1チャット
- 連携：なし
- システム設定：なし
